Trabalho de CSS - Camadas de Aprendizado

Camada 1 – Conceitos Básicos

CSS: Cascading Style Sheets (Folhas de Estilo em Cascata).

Função: O CSS serve para estilizar páginas web, mudando cores, tamanhos, posições e aparência geral.

HTML vs CSS: HTML estrutura o conteúdo, enquanto o CSS estiliza esse conteúdo.

Formas de usar CSS:

  • Inline (dentro da tag)
  • Interno (na tag style)
  • Externo (arquivo .css separado)

Camada 2 – Propriedades CSS

Propriedade Significado Exemplo
color Define a cor do texto color: red;
background-color Define cor de fundo background-color: blue;
font-size Tamanho da fonte font-size: 18px;
font-family Tipo da fonte font-family: Arial;
text-align Alinhamento do texto text-align: center;
width Largura width: 300px;
height Altura height: 200px;
border Borda do elemento border: 1px solid black;
margin Espaço externo margin: 10px;
padding Espaço interno padding: 10px;

Camada 3 – Organização e Layout

Seletor CSS: É o elemento que escolhe qual parte do HTML será estilizada.

Class: Serve para aplicar estilos em vários elementos.

ID: Serve para identificar um único elemento.

Diferença: Class pode repetir, ID é único.

Box Model: Modelo que define margem, borda, padding e conteúdo.

Camada 4 – Acessibilidade

O CSS ajuda pessoas com deficiência visual ao melhorar contraste e legibilidade.

O contraste é importante para facilitar a leitura.

Fontes grandes ajudam na leitura e acessibilidade.

  • Usar bom contraste
  • Usar tamanhos de fonte legíveis
